# peppol-outbound [![CircleCI](https://circleci.com/gh/OpusCapita/peppol-outbound.svg?style=svg)](https://circleci.com/gh/OpusCapita/peppol-outbound)

Peppol OpusCapita Access Point outbound service running on Andariel Platform.

The service uses Oxalis to send documents. It first sends a Lookup request and then sends the actual file. 

Please check the wiki pages for more information:

* [Supported Formats](https://opuscapita.atlassian.net/wiki/spaces/IIPEP/pages/107806851/Supported+document+types+profiles+and+formats)
* [Outbound Sending Errors](https://opuscapita.atlassian.net/wiki/spaces/IIPEP/pages/107806861/Outbound+sending+errors)
